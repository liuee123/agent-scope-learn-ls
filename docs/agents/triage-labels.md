# Triage Labels

技能使用五个标准 triage 角色来沟通。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| 技能中的标签        | 实际标签             | 含义                               |
| ------------------- | -------------------- | ---------------------------------- |
| `needs-triage`      | `needs-triage`       | 维护者需要评估此 issue             |
| `needs-info`        | `needs-info`         | 等待报告者提供更多信息             |
| `ready-for-agent`   | `ready-for-agent`    | 已充分细化，可由 AFK agent 处理    |
| `ready-for-human`   | `ready-for-human`    | 需要人工实现                       |
| `wontfix`           | `wontfix`            | 不会处理                           |

当技能提到某个角色（如"应用 AFK-ready triage 标签"），请使用本表中对应的标签字符串。

如需修改，编辑右侧列以匹配你实际使用的词汇。
